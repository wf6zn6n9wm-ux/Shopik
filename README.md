# Shopik

Прототип платформи Shopik — магазин з Instagram/Telegram за 5 хвилин.

Зібрано на Vite + React + Tailwind (CDN).

## Запуск локально
```
npm install
npm run dev
```

## Деплой
Підключи репозиторій до Vercel — збирання автоматичне (`npm run build`).

## Ігри 🎮
- `durak.html` — класичний **підкидний Дурак** (1 на 1 проти бота, колода 36 карт,
  козир, підкидування, «бито»/«взяти», добір до 6 карт).
- `game.html` — міні-аркада **Shopik Catch**: лови замовлення у корзину, оминай бомби.

Обидві ігри — самодостатні файли без залежностей: просто відкрий у браузері (працює і на телефоні).
