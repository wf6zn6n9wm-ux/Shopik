# Shopik

Прототип платформи Shopik — магазин з Instagram/Telegram за 5 хвилин.

Зібрано на Vite + React + Tailwind (CDN).

## Запуск локально
```
npm install
npm run dev
```

## Деплой
Підключи репозиторій до Vercel — збирання автоматичне (`npm run build`).

## Гра 🎮
`game.html` — міні-аркада **Shopik Catch**: лови замовлення у корзину, оминай бомби.
Один самодостатній файл без залежностей — просто відкрий у браузері (працює і на телефоні).
